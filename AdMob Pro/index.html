AdMob Pro Plugin Demo

The AdMob Pro plugin offers an easy way to show Google AdMob banners in your app. You can choose from various banner sizes and positions.

There is also a non-Pro version which is no longer maintained by the author.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/Windows Phone - App package.
